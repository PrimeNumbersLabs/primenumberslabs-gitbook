# Chainlink

### <mark style="color:purple;">Overview</mark>

**Chainlink** provides the decentralized price oracle infrastructure used by **PrimeFi** to calculate:

* Asset values
* Collateralization ratios

***

### <mark style="color:purple;">Why It Matters</mark>

Chainlink Price Feeds aggregate data from multiple premium sources, ensuring:

* Accurate pricing in real time
* Reliability during high market volatility

This is critical for a lending protocol, where:

* Pricing directly affects liquidation thresholds
* Incorrect data can impact user safety

***

### <mark style="color:purple;">Role in PrimeFi</mark>

Within PrimeFi, Chainlink enables:

* Secure valuation of collateral assets
* Precise loan-to-value (LTV) calculations
* Reliable liquidation mechanisms

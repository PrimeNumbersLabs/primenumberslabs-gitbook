# LayerZero

### Overview

**LayerZero** is the omnichain messaging protocol that powers:

* PrimeFi’s cross-chain architecture
* The **$PRFI OFT standard**

It enables asset and data transfers across **70+ blockchains** without relying on:

* Centralized bridges
* Wrapped tokens
* Middlechains

This is achieved through its **Ultra Light Node (ULN)** infrastructure, enabling secure and gas-efficient cross-chain communication.

***

### How Prime Numbers Uses LayerZero

Prime Numbers integrates LayerZero across multiple layers of the ecosystem:

* PrimeFi: cross-chain collateral and loan management
* $PRFI OFT standard: unified token supply across all chains
* PRFI NFTs (ONFT standard): omnichain NFT transfers (in development)

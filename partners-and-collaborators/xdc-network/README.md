# XDC Network

### Overview

**XDC Network** is the primary blockchain on which Prime Numbers was built.

It is a hybrid blockchain that combines **public and private ledger capabilities**, using a **Delegated Proof of Stake (XDPoS)** consensus mechanism compatible with EVM.

***

### Key Characteristics

* 2,000 transactions per second
* Near-zero transaction fees
* EVM compatibility
* Focus on Trade Finance, ISO 20022, and institutional financial services

***

### Strategic Partnership

The partnership with XDC Network was established in **2022**, when Prime Numbers Labs moved to Dubai.

* Arturo Cantera (CEO & Founder) joined the XDC Network core team
* Prime Numbers became the **primary DeFi infrastructure builder** for the network

***

### Ecosystem Contribution

Prime Numbers has developed the full public DeFi stack on XDC, including:

* Liquid staking
* Lending and borrowing (PrimeFi)
* NFT marketplace (PrimePort)
* Omnichain integrations (LayerZero, Stargate, Etherscan)
* Validator infrastructure

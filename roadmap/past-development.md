# Past Development

### 2021: Genesis

The original concept for **Prime Numbers** was born.

* Initial token sale completed
* First staking solutions developed:
  * PRFI Staking v1
  * PRFI Staking v2

***

### 2022: Building the Foundation

* PrimePort v0: first NFT marketplace on XDC Network launched
* XDC Staking v0: initial staking system introduced
* Strategic partnership with XDC Network
  * Arturo Cantera joins XDC core team

***

### 2023: Expanding and Innovating

* PRFI Staking v3: enhanced features and reward mechanics
* PrimePort v1 & v2: major marketplace upgrades
* PrimeFi v0: first version of the lending and borrowing protocol
* XDC Staking v1 & v2: continued improvements

***

### 2024: Infrastructure and Token Rebuild

#### Q1–Q2 2024

* PRFI Staking v3 and XDC Staking v2
* Staking Interface v3

#### Q3 2024

* Tokenomics rebuild
* Vaults and liquid staking architecture introduced
* PrimeFi Testnet launch

#### Q4 2024

* PRNT → PRFI token migration
* PrimeFi incentivized testnet campaign

***

### 2025: Launch and Expansion

#### Q1 2025

* $1.2M private round raised from angel investors
* PRFI adopted as OFT standard (LayerZero)
* PRFI public sale and Token Generation Event (TGE)
* PRFI listed on Hyperliquid L1
* Full audits completed:
  * PrimeFi (PeckShield)
  * $PRFI (QuillAudits)
  * XDC Staking (QuillAudits)
* PRFI NFTs airdrop
* PrimeFi Mainnet launch
* Immunefi bug bounty program announced

#### Q2 2025

* PrimeFi new interface release
* Integration with PRFI NFTs
* Hyperliquid HyperEVM integration

#### Q3 2025

* PRFI NFT Boosts activation
* PrimeStaking interface with analytics

#### Q4 2025

* Integration with XDC Network (deployment and production)
* Launch of in-house analytics (TVL, Borrow, Utilization, Users)
* Custody contracts audited by Nethermind Security

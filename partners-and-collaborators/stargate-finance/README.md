# Stargate Finance

#### <mark style="color:purple;">Overview</mark>

**Stargate Finance** provides the omnichain liquidity infrastructure used by **PrimeFi**.

Built on top of LayerZero, it enables native asset transfers across chains without:

* Wrapped tokens
* Liquidity fragmentation

***

#### <mark style="color:purple;">How It Works</mark>

Stargate uses unified liquidity pools to facilitate:

* Native cross-chain asset transfers
* Instant settlement across supported networks

This allows users to move assets seamlessly between chains while maintaining native liquidity.

***

#### <mark style="color:purple;">Role in PrimeFi</mark>

Within PrimeFi, Stargate enables:

* Bridging of major assets across chains, including:
  * USDC
  * USDT
  * ETH
  * BTC
  * OFTs
* Direct integration into the PrimeFi interface
* Cross-chain capital efficiency without additional steps

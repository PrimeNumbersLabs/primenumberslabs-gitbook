# Introduction to PRFI

$PRFI is the utility token of the Prime Numbers ecosystem.\
It connects PrimeFi, PrimeStaking, and PrimePort into a single economic\
system, where protocol revenue flows back to token holders through a\
programmable NFT staking mechanism.

$PRFI is an Omnichain Fungible Token (OFT) built on LayerZero's OFT standard.\
This means it moves natively across Ethereum, Base, XDC, Hyperliquid, and\
Arbitrum without wrapped token mechanics, separate liquidity pools, or bridge risk.\
A single unified supply exists across all supported chains.

Total supply: 100,000,000 $PRFI, fixed, no inflation.
